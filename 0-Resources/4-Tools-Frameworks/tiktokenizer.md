## Summary

TikTokenizer is an interactive web tool for visualizing and understanding how
LLM tokenizers work. It supports multiple encoders (GPT2, cl100k_base, o200k_base, etc.)
and lets you input text to see how it gets split into tokens with their indices.

## Use Cases

- Understanding BPE tokenization visually
- Comparing encodings across different tokenizer versions
- Debugging token count and splitting behavior in prompts
