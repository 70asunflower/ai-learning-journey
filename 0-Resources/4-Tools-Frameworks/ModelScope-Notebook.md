# ModelScope Notebook 免费试用

> ModelScope（魔搭社区）提供的云上 Notebook 开发环境，开箱即用，包含免费 GPU/CPU 资源。
> 官方文档: https://www.modelscope.cn/docs/notebooks/free-trial

## Overview

ModelScope Notebook 是一个开箱即用的云上开发环境，基于 GPU 和 CPU 资源，预装 ModelScope 开发镜像，为社区注册用户提供免费使用时长。

## 核心特性

- **开箱即用**: 预装开发环境镜像，无需手动配置依赖
- **免费时长**: 注册用户可免费使用一定时长
- **GPU 支持**: 支持 GPU 加速推理/训练
- **CPU 实例**: 也提供 CPU 实例供轻量使用
- **云上体验**: 浏览器直接开发，无需本地环境

## ModelScope 平台概览

ModelScope（魔搭社区）是阿里达摩院开源的 AI 社区平台，提供一站式 AI 开发服务：

| 功能 | 说明 |
|------|------|
| **模型库** | 开源模型托管与发现（NLP、CV、语音、多模态等） |
| **数据集** | 各领域数据集托管 |
| **Studio（创空间）** | 基于 Gradio 的 AI 应用开发平台 |
| **Notebook** | 免费 GPU/CPU 云上开发环境 |
| **Library** | Python 库，统一加载各域模型 |
| **模型训练与推理** | 基于 SWIFT 框架的大模型训练/微调 |
| **模型服务** | 云端模型部署与推理服务 |
| **评测（司南）** | 模型能力评测与基准测试 |

## 使用场景

- 快速验证开源模型（推理、测试）
- 轻量微调/训练实验
- 学习 AI 技术栈
- 部署在线 Demo
