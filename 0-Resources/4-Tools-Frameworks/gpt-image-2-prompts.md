## 提示词编写技巧

### 1. JSON 结构化描述
大部分高质量提示词采用 JSON 格式，将图像分解为结构化层级：
- `type` — 图像类型
- `style` — 整体视觉风格
- `theme` — 主题/配色
- `layout` — 布局结构
- `sections` — 各区域详细定义
- `header` / `footer` — 顶部/底部元素

### 2. 参数化变量
使用 `{argument name="..." default="..."}` 实现模板化：
```
{argument name="city name" default="成都"}
{argument name="product name" default="Meta Quest 3"}
{argument name="primary accent color" default="霓虹紫"}
```
一套提示词快速适配不同主题，只需替换参数值。

### 3. 像素级文字渲染控制
精确指定标题、按钮文案、标签/徽章、价格、统计数据、聊天消息、图例标签。

### 4. 布局精确描述
明确定义各元素的空间位置：
- `"position": "左上"` / `"右上"` / `"左下"` / `"右下"`
- 网格结构：`"structure": "4 个等分象限"`

### 5. 组件化设计思维
将复杂图像拆解为独立组件：卡片、图标、按钮、进度条、数据可视化、导航栏。

### 6. 自然语言叙述式（适用于艺术插画）
非 UI 类图像（如动漫场景）采用自然语言叙述：精确描述角色外观、动作与能量效果、环境背景、光影与氛围。

### 7. 引用参考图
```
"instruction": "以 REFERENCE_0 作为结构基础，将平面矢量设计转化为高度逼真的 3D 信息图。"
```
