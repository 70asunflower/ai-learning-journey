# JAX Scaling Book

## 核心要点（摘要）

JAX 官方团队出品的 LLM Scaling 实战书籍，系统讲解如何用 JAX/TPU 进行大规模模型训练的分布式扩展。

## 详细内容

- 覆盖 LLM 训练的 Scaling 全链路：数据并行、模型并行、流水线并行
- JAX 生态下的分布式训练最佳实践
- 适合理解大模型训练的系统工程层面

## 个人备注

推理学习路线中的模型层参考。理解 scaling 才能理解推理优化的输入端——大模型是怎么训出来的，哪些结构特性影响推理效率。
