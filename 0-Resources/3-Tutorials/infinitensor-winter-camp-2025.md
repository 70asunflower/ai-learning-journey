# Infinitensor 夏立营 2025

## 核心要点（摘要）

推理基础设施全栈训练营，覆盖推理 infra 三层架构：模型层 → 框架层 → 算子层。推荐作为推理方向系统学习的入口。

## 详细内容

- **模型层**：大模型结构分析、推理特性、量化/剪枝等模型侧优化
- **框架层**：推理框架设计（对应 nano-vllm / mini-sglang 这类项目）
- **算子层**：底层算子优化（对应 FlashAttention v1/v2/v3 这类工作）
- 实战建议：三层都了解，选其中一层深耕
  - 框架层 → nano-vllm / mini-sglang
  - 算子层 → FlashAttention v1/v2/v3

## 个人备注

推理 infra 的"全景图"训练——三层架构的划分非常清晰，帮助定位自己想深耕的方向。推荐先过一遍夏立营的整体内容建立框架认知，再选某一层深入实战。
