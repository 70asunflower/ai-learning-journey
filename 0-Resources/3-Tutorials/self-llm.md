# self-llm: 开源大模型食用指南

> 中文大模型教程项目，帮助初学者掌握开源大模型的部署、使用和微调。
> 官网: https://github.com/datawhalechina/self-llm

## Overview

由 DataWhale 社区维护的开源大模型教程项目，面向国内初学者，基于 Linux 平台。覆盖环境配置、部署使用、高效微调的完整链路。

## Content Structure

1. **环境配置** — 针对不同模型提供详细的环境安装步骤
2. **部署使用** — LLaMA、ChatGLM、InternLM、Qwen 等模型的部署与调用
3. **应用集成** — 命令行调用、在线 Demo 部署、LangChain 集成
4. **微调训练** — 全量微调、LoRA、P-Tuning、分布式微调

## Models Covered

- Qwen1.5 / Qwen2
- InternLM / InternLM2
- MiniCPM
- LLaMA 系列
- ChatGLM 系列
- 及其他开源模型

## My Notes

- **学习顺序建议**: 环境配置 → 部署使用 → 微调训练
- **初学者推荐模型**: Qwen1.5, InternLM2, MiniCPM
- **价值**: 中文生态友好的教程，步骤清晰，适合快速上手实践
