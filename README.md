# AI Learning Journey

记录AI学习过程中的资源收集、学习笔记和项目实践。

## 目录结构

```
ai-learning-journey/
├── 0-Resources/          # 资源分类收藏
│   ├── 1-Official-Docs/  # 官方文档、最佳实践
│   ├── 2-Papers/         # 重要论文
│   ├── 3-Tutorials/      # 优质教程、博客
│   ├── 4-Tools-Frameworks/  # 框架、工具
│   └── 5-Datasets/       # 数据集
├── 1-Notes/              # 个人学习笔记
│   ├── 1-Foundations/    # 基础概念
│   ├── 2-Models/         # 模型相关笔记
│   ├── 3-Training/       # 训练技巧
│   └── 4-Deployment/     # 推理部署
├── 2-Code/               # 可运行代码示例
├── 3-Projects/           # 综合项目
└── scripts/              # 辅助脚本
```

## 工作流

1. **收藏**: 发现新资源 → 添加到 `0-Resources/` 对应分类
2. **学习**: 阅读消化 → 在 `1-Notes/` 记录心得
3. **实践**: 运行验证 → 整理到 `2-Code/`
