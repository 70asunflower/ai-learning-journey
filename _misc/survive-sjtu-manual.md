# 上海交通大学生存手册 (SurviveSJTUManual)

> 交大学生写给交大学生的非官方生存指南，涵盖心态建设、留学保研、选课备考、科研方法等。
> 官网: https://survivesjtu.gitbook.io/survivesjtumanual
> GitHub: https://github.com/SurviveSJTU/SurviveSJTUManual

## Overview

由上海交通大学学生协作编写维护的"生存手册"，源于 2008 年一群交大本科生编写的原始版本，后因原版主要面向出国读博群体而在国内深造、就业等方面有所欠缺，因此社区持续更新扩充。风格务实直白，带有"民间智慧"色彩。

## Content Structure

### 一、立志篇
- 批判"高考思维"和"被动思维"，反对 PUA
- 引导思考个人价值、研究方向和职业规划
- 核心文章：
  - **你想要做什么** — 建立自主意识
  - **失败的思维方式** — 高考思维 / 被动思维
  - **反对PUA** — 识别和抵抗学术 PUA
  - **你的身价是多少** — 客观认识自我价值
  - **正确地浪费剩下的时间** — 时间投入的 ROI 思维
  - **做研究的兴趣** — 科研动机的自我审视

### 二、访谈集
- **为了留学而出国** — 为什么出国、读什么学位、申请步骤、警惕中介
- **做真正的研究** — 研究态度、区分上学与研究、知识积累、学术价值衡量
- **管理者的智慧** — 御人先御己、培养团队、进度控制
- **小心项目的陷阱** — 辨别项目质量、选对导师、团队与合作伙伴
- **保研者说** — 明确目的、需要做些什么、硕士之后
- **破解留沪政策** — 户口心态、审批制度、积分规则

### 三、生存技巧
- **转专业指南** — 交大转专业流程与经验
- **选课原则与选课技巧** — 是否选二专、选课选什么、广占坑策略
- **突击备考** — 复习范围、三天冲刺法、复习箴言
- **正确解读成绩算法** — GPA 制度概览、选修课利用、理性挂科与重修
- **旁门左道** — 免听制度、形势政策考试、物理实验应对

### 四、附录（部分建设中）
- 本科生专业介绍、转专业/转学/找实习/科研经验分享
- 社团介绍、通识课推荐、已工作学长经验

## Key Takeaways

1. **主动思维 > 被动执行**：批判"高考思维"（排名驱动）和"被动思维"（等安排），建立自主决策意识
2. **时间 ROI**：不是所有"努力"都有价值，正确地"浪费"时间比低效勤奋更重要
3. **选导师 > 选项目**：项目可以换，导师换不了；识别学术 PUA 是保命技能
4. **GPA 是工具不是目的**：选修课拉 GPA、理性挂科重修，都是合法策略
5. **出国/保研/就业 三条路的核心差异**：早想清楚比什么都重要

## Relevance

虽然面向交大学生，但心态建设和策略思维对任何研究生都有参考价值：
- 反对 PUA / 识别有毒导师 — 通用
- 主动 vs 被动思维 — 通用
- 选课/GPA 策略 — 可参考思路
- 留学申请/保研准备 — 可参考框架
