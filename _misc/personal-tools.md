# Stickies

> Windows 桌面便签工具，支持富文本、定时提醒、自动关闭等高级功能。
> 官网: https://www.zhornsoftware.co.uk/stickies/index.html

## Overview

Stickies 是 Windows 平台的一款免费桌面便签工具，由 Zhorn Software 开发。它比系统自带的便签功能强大得多，支持丰富的格式、提醒、导出导入等功能。

## Key Features

- **桌面便签**: 在桌面任意位置放置便签，支持自定义颜色、字体、大小
- **富文本**: 支持粗体、斜体、链接、图片插入等
- **定时关闭**: 可设置便签在指定时间自动关闭，适合做临时提醒
- **定时显示**: 可设置便签在指定时间自动弹出
- **导出/导入**: 支持 JSON 格式导出导入，方便备份
- **置顶/钉住**: 便签可始终置顶或钉在桌面，不被其他窗口遮挡
- **搜索**: 支持在便签内容中全文搜索
- **热键**: 可设置全局热键快速创建便签

## Use Cases

- 临时笔记、快速记录灵感
- 会议提醒、待办清单
- 日常学习重点标记
- 临时网址、代码片段保存

## Notes

- **免费软件**: 个人使用免费，商业使用需付费
- **仅 Windows**: 不支持 macOS/Linux
- 替代方案（Mac）: macOS 自带的 Stickies 应用，或第三方应用如 NotePlan、Notion
